# Gate: G-18 — E1b EOF flush (3B)

## SSOT anchors
- spec/ssot.yaml: e1b_streaming_accum_emit.line_termination_and_flush.eof_flush_sealed

## Goal
파일 끝에 `\n`이 없더라도, 마지막 라인에 유효 음절이 남아 있으면 1워드를 방출함을 증명한다.

## Checks (must pass)
- [ ] 입력이 `...<valid syllable>`로 끝나고 개행이 없을 때도 출력이 4바이트 생성된다.
- [ ] 마지막 라인이 공백/탭/CR만 있고 개행이 없으면 출력은 0바이트다.

## Test vectors
- 케이스1: 유효 음절 1개, EOF(개행 없음) → 4B
- 케이스2: 공백만, EOF → 0B
- 케이스3: 유효 음절 + 공백 + EOF → 4B
