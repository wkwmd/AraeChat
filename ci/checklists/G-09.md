# Gate: G-09 — Exit code policy (0 success, 1 any failure)

## SSOT anchors
- spec/ssot.yaml: cli_process_contract.exit_codes.success
- spec/ssot.yaml: cli_process_contract.exit_codes.any_failure

## Goal
성공은 0, 모든 실패는 1로 통일됨을 증명한다.

## Checks (must pass)
- [ ] 대표 성공 케이스에서 exit == 0
- [ ] 대표 실패 케이스(잘못된 argv, SyntaxError, I/O 실패 등)에서 exit == 1
- [ ] stderr_len == 0 (SSOT 공통)

## Test vectors
- 성공: 유효 입력 파일 + 쓰기 가능한 out → exit 0
- 실패1: argv arity 오류 → exit 1
- 실패2: 금지 문자 포함 → exit 1
- 실패3: 입력 파일 없음 → exit 1
